Sobre

Sou Eduardo dos Santos Ferreira, profissional de TI com atuação em desenvolvimento de software, automação de processos e suporte técnico. Trabalho na WG Convênios como Técnico de TI, onde sou responsável pela manutenção de infraestrutura, otimização de sistemas internos e garantia de estabilidade operacional. Paralelamente, colaboro na Ajuda Cidadão como Desenvolvedor, contribuindo para o desenvolvimento e aprimoramento de soluções web utilizadas diariamente pela equipe.

Minha experiência integra desenvolvimento web, criação de aplicações desktop em Python e integração entre sistemas, sempre com foco em eficiência, confiabilidade e melhoria contínua. Unifico conhecimento técnico de software e infraestrutura para entregar soluções completas — da análise à implementação.

Projetos de Software

CleanCrow

O software para limpeza, otimização e atualização do sistema operacional Windows.

Crowvert

Crowvert é um software com interface gráfica via Tkinter/PyQt5, desenvolvido para facilitar a conversão de arquivos em massa.

DarkFeather

DarkFeather é uma ferramenta gráfica desenvolvida em Python com Tkinter para visualizar e extrair informações detalhadas das redes Wi-Fi salvas no sistema Windows.

BlackWing

O killer de processos do Windows: mate qualquer processo travado com facilidade e segurança.

Ravenizer

About Ravenizer é uma ferramenta de atualização automatizada para Windows que utiliza winget e Chocolatey para manter seu sistema e aplicativos sempre atualizados com um único clique.

BruteRaven

BruteRaven é um projeto educacional que demonstra como realizar força bruta em redes Wi-Fi, com interface gráfica feita em Tkinter e motor de execução usando pywifi.

Coloeus

Coloeus é uma ferramenta para Windows que oferece uma interface para gerenciamento de programas instalados, análise de arquivos, uso de disco e controle de permissões administrativas.
